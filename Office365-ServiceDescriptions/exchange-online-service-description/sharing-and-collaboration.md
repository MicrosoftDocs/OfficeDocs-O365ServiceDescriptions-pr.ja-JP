<span data-ttu-id="647ef-122">Office 365 のプランは SharePoint を含む必要があります。サイト メールボックスでは、ユーザーは SharePoint と Exchange の両方のライセンスを必要とします。</span><span class="sxs-lookup"><span data-stu-id="647ef-122">For more information about site mailboxes, see [Site Mailboxes](https://go.microsoft.com/fwlink/p/?LinkId=271789).</span></span>
