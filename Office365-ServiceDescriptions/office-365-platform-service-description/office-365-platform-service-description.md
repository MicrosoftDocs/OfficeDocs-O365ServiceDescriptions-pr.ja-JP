# <a name="microsoft-365-and-office-365-platform-service-description"></a>Microsoft 365 および Office 365 プラットフォーム サービスの説明

## <a name="multi-geo-capabilities"></a>複数地域の機能 

複数地域を使用すると、1 つの組織が複数の Office 365 データセンターの地域にまたがると、Exchange Online と OneDrive for Business のデータをユーザー単位で、選択した地域に保存できます。 
  
利用可能な地域: アジア太平洋、オーストラリア、カナダ、欧州連合、インド、日本、韓国、英国、米国。 
  
詳細については、「複数地域の導入」[を参照Office 365。](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Introducing-Multi-Geo-in-Office-365/ba-p/107016)
