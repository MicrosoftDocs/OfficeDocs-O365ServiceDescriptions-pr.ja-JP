# <a name="project-online-service-description"></a>Project Online サービスの説明

オンラインのプロジェクトは、プロジェクト ポートフォリオ管理 (PPM) および日常の作業の柔軟なオンライン ソリューションです。Office 365 経由で配信、オンライン プロジェクトは、計画、優先順位を付け、およびプロジェクトとプロジェクトのポートフォリオへの投資を管理するための強力なプロジェクト管理機能、ほぼどこからでもほとんどすべてのデバイスにします。オンラインのプロジェクト管理者、ポートフォリオ管理者、閲覧者、プロジェクトおよびリソース マネージャーとチームの責任者およびメンバーを使用できます。 
  
Project Online は、Project Online Essentials、Project Online Professional、および Project Online Premium の 3 つのプランで使用できます。
  
各プランの機能を比較するには、「[プランと価格](https://products.office.com/en-us/project/compare-microsoft-project-management-software)」を参照してください。
  
Office 365 サービスの説明のページは、エクスポート、保存、印刷が可能です。[複数のページをエクスポート](https://go.microsoft.com/fwlink/?LinkId=403349)する方法について説明します。
  
## <a name="project-online-boundaries-and-limits"></a>Project Online の境界と制限

Project Online サービスにはいくつかの制約事項があります。詳細については、「[Project Online: ソフトウェアの境界と制限 ](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489)」をご覧ください。
  
## <a name="data-backup"></a>データ バックアップ

 Project Online は SharePoint Online 上にビルドされ、SharePoint と同じデータ バックアップ ポリシーに従います。詳しくは、「[SharePoint Online サービスの説明](https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx)」にある「データ バックアップ」セクションをご覧ください。
  
## <a name="use-of-sharepoint-online"></a>SharePoint Online の使用

プロジェクトのオンライン サービスには、プロジェクトのオンライン サービスの一部としてユーザのプロビジョニングは、SharePoint Online での使用が必要です。プロジェクト オンライン プロフェッショナルとプレミアム SharePoint Online の機能に対する権限は、保存して、オンラインのプロジェクトをサポートするデータへのアクセスに制限されます。
  
## <a name="licensing-considerations"></a>ライセンスに関する考慮事項

### <a name="internal-users"></a>内部ユーザー

プロジェクトのオンライン サイトでの操作には、プロジェクトのオンラインのライセンスが必要です。
  
- Project Online Essentials が最低限必要です。
    
- 高度なユース ケース、ユーザーを適切なライセンスを持っている必要があります。プロジェクト マネージャーは、プロジェクトのオンライン プロフェッショナル ライセンスを最低限必要があります。ポートフォリオ マネージャーには、プロジェクトのオンラインのプレミアム ライセンスが必要です。
    
> [!IMPORTANT]
>  プロジェクト オンラインに関する重要事項は、オンラインの専門家をプロジェクトまたはプロジェクトのオンラインのプレミアム顧客のチーム メンバーの*アドオン*ライセンスです。<br/>最後のオンラインの専門家をプロジェクトまたはプロジェクトのオンラインのプレミアム ライセンスが終了すると、120 日後、Project Web App インスタンスが削除されます。<br/>プロジェクト オンラインの試用版サブスクリプションに、Project Web App インスタンスが削除されます、試用期間終了後 30 日間。 
  
### <a name="external-users"></a>外部ユーザー

プロジェクト オンラインの外部のユーザーのライセンスは、内部ユーザー用と同じポリシーに従います。プロジェクトのオンライン サイトでの操作には、プロジェクトのオンラインのライセンスが必要です。
