# <a name="excel-for-the-web"></a>Web 用 Excel

Excel for the web (旧Excel Web App) は、Microsoft Excel エクスペリエンスを Web ブラウザーに拡張し、ブックが保存されている Web サイトでブックを直接使用できます。 すべてのお客様は、Web 用のアプリを使用OfficeファイルOffice簡単に編集できます。
  
> [!NOTE]
> 50 メガバイト (MB) を超えるブックは、Microsoft SharePoint Online 内から Excel for the web で表示できません。 これよりサイズの大きいファイルを表示するには、ご使用のコンピューターに Excel デスクトップ アプリをインストールする必要があります。 

> [!NOTE]
> オファーを比較するために、この記事に記載されている高度な機能には、Office スイートでのみ使用できる Excel デスクトップ アプリ機能 (Office Professional Plus、Office Standard、Microsoft 365 Apps for enterprise など) が含まれます。 Word for the web や PowerPoint for the web など、Web 機能のすべての Office の一覧については、「Web プランの Office 全体の機能の可用性」を [参照してください](office-online-service-description.md#feature-availability-across-office-for-the-web-plans)。

Excel for the web でサポートされているファイルの種類を確認するには、「サポートされているファイルの種類」を参照Office [を参照してください](office-online-service-description.md#supported-file-types-for-office-for-the-web)。 Web 用 Excel と Excel デスクトップ アプリの違いの詳細については、「ブラウザーでブックを使用する場合と Excel でブックを使用する場合の相違点」を [参照してください](https://support.office.com/article/f0dc28ed-b85d-4e1d-be6d-5878005db3b6)。
  
## <a name="add-background-color-to-sheet-tabs"></a>シート タブに背景色を追加する 

シート タブに背景塗りつぶしの色をブックの下部に追加して目立たせる。詳細については、「シート タブ [に背景色を追加する」を参照してください](https://support.office.com/article/440B28F2-3146-4DCA-95DF-3B9D43ACBE59)。

## <a name="add-hyperlinks"></a>ハイパーリンクの追加

セルにハイパーリンクを追加することによって、Web ページや Web 上のファイルにリンクします。セルに直接 Web アドレスを入力するか、または [ハイパーリンクの挿入] コマンドを使用することができます。

## <a name="advanced-data-types-stocks-and-geography"></a>高度なデータ型: 株式と地理

Excel で在庫データと地理データを取得できます。 セルにテキストを入力し、それを Stocks データ型または Geography データ型に変換するのと同じほど簡単です。 株式と [地理データ型の詳細については、以下を参照してください](https://support.office.com/article/e61a33056-9935-484f-8ac8-f1a89e210877)。
  
## <a name="advanced-time-filtering-timeline-slicer"></a>Advanced time filtering (Timeline slicer)

タイムライン コントロールでは、モデル化されたデータのフィルター処理を行うことができます。この機能を使用すると、連続した日付範囲の表示や変更、ピボットテーブルやピボットグラフなどのピボットベースのオブジェクトのフィルター処理を視覚的に行うことができます。  
  
## <a name="alignment"></a>Alignment

[ホーム] タブの配置ボタンを使用して、テキストの配置を変更します。  
  
## <a name="apply-conditional-formatting"></a>条件付き書式の適用

条件付き形式を使用して、データを視覚的に調査および分析し、重大な問題を検出し、パターンと傾向を特定するのに役立ちます。 Excel for the web で条件付 [き書式を使用する方法について詳しくは、以下を参照してください](https://support.microsoft.com/office/use-conditional-formatting-to-highlight-information-fed60dfa-1d3f-4e13-9ecb-f1951ff89d7f?)。
  
## <a name="apply-data-validation-to-cells"></a>Apply data validation to cells

この機能は、Web 用の Excel で使用できます。 その他の高度な機能は、Excel デスクトップ アプリでのみ使用できます。
  
## <a name="apply-smartrecommended-formatting"></a>Apply smart/recommended formatting

Web 用の Excel では使用できません。 Excel デスクトップ アプリでは、推奨グラフを使用して、データの表示に最適なさまざまなグラフを選択できます。 Excel でグラフ [の推奨事項を使用する方法について詳しくは、以下を参照してください](https://support.office.com/article/cd131b77-79c7-4537-a438-8db20cea84c0)。
  
## <a name="autocomplete"></a>オートコンプリート

Excel for the web では、セルに入力するデータを完了できます。また、複数の可能な値がある場合は、選択できるリストにすべて表示します。
  
## <a name="autosum"></a>AutoSum

列または行の数値を合計するには、合計するセルを選択して [オート SUM] をダブルクリックします。結果は次の空白のセルに表示されます。
  
## <a name="calculations"></a>計算

Excel を使用して、さまざまな自動、手動、反復計算を実行することができます。Excel では、これらの計算を実行するのに関数を使用します。詳細については、「[Excel 関数 (カテゴリ別)](https://go.microsoft.com/fwlink/?linkid=847459)」を参照してください

## <a name="cell-references-in-formulas"></a>数式のセル参照

## <a name="chart-animations-adapt-to-new-data"></a>新しいデータに合わせたグラフ アニメーション
高度なグラフ機能は、Microsoft Excel デスクトップ アプリのみでサポートされます。 [SmartArt グラフィックをアニメーション化する方法の詳細については、次のページを参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271668)。
  
## <a name="charts-and-tables-including-pivotchart-reports-and-pivottable-reports"></a>ピボットグラフ レポートとピボットテーブル レポートを含む、グラフとテーブル

Excel のグラフとテーブルを使えば、データの書式設定と情報の管理を簡単に行うことができます。 詳細については、「Create [and format tables](https://support.office.com/article/e81aa349-b006-4f8a-9806-5af9df0ac664) and [Overview of PivotTable and PivotChart reports」を参照してください](https://go.microsoft.com/fwlink/?linkid=55417)。

## <a name="check-spelling"></a>スペル チェック

ワークシート上のテキストのスペル チェックを行う場合は、[**校正** のスペル チェック  >  **を確認する]**  >  **を選択します**。 スペル チェックの [詳細については、次の情報を参照してください](https://support.office.com/article/185efb8d-ee3f-4511-b7ef-35599c924159)。

## <a name="comment-creation"></a>Comment creation

Web 用の Excel でコメントを挿入、編集、および削除できます。 コメント スレッドを解決して、編集や新しいコメントに閉じても、その場所に残します。 これにより、必要に応じてスレッドをコメントに再度開きます。 [コメント@mentionsを使用して](https://support.office.com/article/644bf689-31a0-4977-a4fb-afe01820c1fd?ad=US#ID0EAADAAA=Online) 、言及したユーザーに電子メールを送信します。 Excel にコメント [を追加する方法について詳しくは、以下を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271670)。

## <a name="convert-a-table-to-a-range"></a>テーブルを範囲に変換する

Excel テーブルを作成した後は、テーブルの機能を使用せずにテーブル スタイルのみを使用できます。 適用した表スタイルの書式を失わずにテーブル内のデータの操作を停止するには、ワークシート内の通常のデータ範囲にテーブルを変換できます。 Excel テーブルをデータ範囲 [に変換する方法について詳しくは、次のリンクを参照してください](https://support.office.com/article/0b326ff1-1764-4ebe-84ea-786265d41c77#ID0EAADAAA=Web)。
  
## <a name="copy-and-paste"></a>Copy and paste

テキスト、ハイパーリンク、数値、数式、図形、グラフ、画像を切り取り、コピー、貼り付けできます。 列内のデータを行に並べ替える必要がある場合は、[貼り付け] メニューの **[トランスポーズ** ] 機能 **を使用** します。 Web 用 Excel [でのコピーと貼り付けについて詳しくは、以下をご覧ください](https://support.office.com/article/682704da-8360-464c-9a26-ff44abf4c4fe#bmk-excelonline)。 ブラウザーとして Edge または Chrome を使用すると、他のアプリからハイパーリンクをコピーし、Web 用の Excel に貼り付けます。 貼り付けハイパーリンクは正常に機能します。
  
## <a name="create-external-data-connections"></a>Create external data connections

Excel for the web を使用してデータ接続を表示できますが、Web 用の Excel を使用して外部データ接続を作成することはできません。 外部データを処理するには、Excel デスクトップ アプリが必要です。 外部データを操作 [する方法の詳細については、「Excel Services」 を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=282330)。 
  
## <a name="create-tables"></a>テーブルの作成

テーブルを作成して関連データの整理と分析を行います。 テーブルを作成することで、データの並べ替え、フィルター処理、および書式設定が容易になります。 スタイル ギャラリーを含む [、テーブルの書式設定オプション](https://support.office.com/article/format-an-excel-table-6789619f-c889-495c-99c2-2f971c0e2370)でいくつかのポリッシュを追加します。
  
## <a name="creation-of-advanced-analysis-views-power-view-power-pivot-slicers"></a>高度な分析ビュー (Power View、Power Pivot、スライサー) の作成

Power Pivot や Power View などの高度な分析機能は、Excel デスクトップ アプリでのみ利用できます。 これらの機能は、Excel for the web ではサポートされていません。 詳細については、「PowerPivot」 [を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271671)。
  
## <a name="data-validation"></a>データ検証

データ検証を使うと、ユーザーがセルに入力するデータや値の種類を制限することができます。 詳細については、「データ検証を [セルに適用する」を参照してください](https://go.microsoft.com/fwlink/?linkid=844735)。
  
## <a name="drag-and-drop-cells"></a>Drag and drop cells

Web の Excel のセル値を移動するには、セルを 1 つの場所から別の場所にドラッグ アンド ドロップします。

## <a name="draw-cell-borders"></a>セルの罫線を描画する

ワークシート上のセルの周囲に境界線を描画して、データを視覚的に整理できます。 Excel のセル [罫線の詳細を参照してください](https://support.office.com/article/dc8a310b-92e3-46a7-9f17-2ab745810f4a#ID0EAADAAA=Web)。
  
## <a name="dropbox"></a>Dropbox

Dropbox は、クラウド ストレージ、ファイルの同期、個人用クラウド、およびクライアント ソフトウェアを提供するファイル ホスティング サービスです。
  
## <a name="duplicate-copy-worksheets"></a>ワークシートの複製 (コピー)

ワークシートは、ブック内または Web 用の Excel の別のブックに複製 (またはコピー) できます。 Excel for [the Web でのワークシートのコピーの詳細を参照してください](https://support.office.com/article/47207967-bbb2-4e95-9b5c-3c174aa69328#ID0EAACAAA=Web)。

## <a name="embed-workbook-on-web-or-blog-page-onedrive"></a>Web またはブログ ページにブックを埋め込む (OneDrive)

OneDrive.com に Excel ブックを保存すると、ブログや Web サイトに直接 Excel ブックを埋め込むことができます。閲覧者はデータの並べ替え、フィルター処理、計算をその場で行うことができます。また、OneDrive のブックを更新すると、閲覧者がページを更新したときに最新の変更内容が表示されます。[ブログへの Excel ブックの埋め込み](https://go.microsoft.com/fwlink/p/?LinkId=271673)の詳細をご確認ください。
  
## <a name="external-references-links"></a>外部参照 (リンク)

外部参照 (リンクとも呼ばれる) とは、別の Excel ブック内のワークシートのセルやワークシートの範囲への参照、または別のブック内の定義された名前への参照のことです。 Excel for the web を使用して外部参照を表示できますが、Web 用の Excel を使用して作成することはできません。 外部参照を作成または更新するには、Excel デスクトップ アプリが必要です。 
  
## <a name="fill-handle"></a>フィル ハンドル

選択したセルまたはセルの範囲の右下に表示されるフィル ハンドルをドラッグして、隣接するセルにデータを入力します。
  
## <a name="find"></a>検索

作業中のワークシートでセルの内容を検索します。 使いやすいダイアログ ボックスでは、ワークシート内の現在の選択範囲から上下に検索できます。 **[すべて検索** ] を使用すると、検索用語のすべてのインスタンスを強調表示できます。
  
## <a name="font-and-cell-formatting"></a>フォントおよびセルの書式設定

必要に応じてデータの表示形式をカスタマイズします。 さまざまなフォント スタイルや色を選択したり、テキストのサイズや色を変更したりできます。 さらに、セルの罫線を描画し、罫線の色を選択できます。
  
## <a name="formula-bar"></a>Formula bar

数式バーでは、セルの表示結果の元になる数式を確認します。 Excel デスクトップ アプリと同じように、ブラウザーで数式の追加、変更、および削除を行うことができます。 Excel for the web では、数式とワークシート上のデータの間に視覚的なマッピングを提供する色で数式の引数が強調表示されます。
  
## <a name="formula-tools-advanced"></a>高度な数式ツール

Microsoft Excel デスクトップ アプリは、3D 参照スタイルなどの高度な数式ツールを提供します。 Web 用の Excel では、動的配列数式などの高度な Excel 数式 [の数が増え続けています](https://support.microsoft.com/office/dynamic-array-formulas-and-spilled-array-behavior-205c6b06-03ba-4151-89a1-87a7eb36e531)。 数式の詳細 [を参照してください](https://support.microsoft.com/office/ecfdc708-9162-49e8-b993-c311f47ca173)。
  
## <a name="freeze-panes"></a>Freeze panes

固定ウィンドウ機能は、Web リボンの Excel で使用できます。 Excel での固定ウィンドウの動作の詳細 [と、Web](https://go.microsoft.com/fwlink/p/?LinkId=282331) [用 Excel のヒントを参照してください](https://go.microsoft.com/fwlink/?linkid=869873)。
  
## <a name="full-fidelity-reading-view"></a>忠実な閲覧表示

編集表示でブラウザーのブックに表示できるものはすべて、閲覧表示でも表示できます。
  
## <a name="functions"></a>関数

Web 用 Excel の数式には、FILTER 関数などの動的配列数式で使用される関数など、400[](https://support.microsoft.com/office/dynamic-array-formulas-and-spilled-array-behavior-205c6b06-03ba-4151-89a1-87a7eb36e531)を超える Excel ワークシート関数のほとんどを[使用できます](https://support.office.com/article/filter-function-f4f7cb66-82eb-4767-8f7c-4877ad80c759)。
  
## <a name="goto"></a>GoTo

GoTo 機能を使用して、スプレッドシート全体をすばやく移動できます。 セル参照を入力すると、スプレッドシートのその場所にジャンプします。 Excel for [the web のキーボード ショートカットの詳細を参照してください](https://go.microsoft.com/fwlink/p/?LinkID=272943)。
 
## <a name="group-data"></a>データをグループ化する

Web スプレッドシートでは、Excel の行と列をグループ化またはアウトラインできます。 キーボード ショートカットを使用すると、作成したグループをすばやく展開または折りたたみできます。 Excel for [the web でのデータのグループ化の詳細について説明します](https://support.office.com/article/08ce98c4-0063-4d42-8ac7-8278c49e9aff#ID0EAADAAA=Web)。
 
## <a name="hideunhide-rows-columns-and-sheets"></a>Hide/unhide rows, columns, and sheets

Excel for the web では、ブック内の行、列、シートを非表示および再表示できます。

## <a name="ideas"></a>アイデア

Excel のアイデアは、高レベルの視覚的な概要、傾向、およびパターンを通じてデータを理解するのに役立ちます。 データ範囲内のセルを選択し、[ホーム] タブの **[アイデア** ] ボタンを **選択** します。Excel のアイデアは、データを分析し、作業ウィンドウでデータに関する興味深いビジュアルを返します。 Excel のアイデア [の詳細を参照してください](https://support.office.com/article/3223aab8-f543-4fda-85ed-76bb0295ffc4)。
  
## <a name="insert-charts"></a>Insert charts

グラフを作成してデータを視覚的に表現します。縦棒グラフ、折れ線グラフ、円グラフ、横棒グラフなどの各種グラフから選択できます。[利用可能なグラフの種類](https://go.microsoft.com/fwlink/p/?LinkId=273899)の詳細をご確認ください。
  
## <a name="insertdelete-rows-and-columns"></a>Insert/delete rows and columns

[ホーム] タブで [**挿入**]または[削除] を選択する前に 1 つ以上の行を選択し、ワークシート内の行または列を一度に 1 行または複数行追加または削除します。

## <a name="insert-pictures"></a>画像を挿入する

スプレッドシートに画像を挿入して、視覚的な関心を追加します。 Web 用に [Excel に画像を挿入する方法の詳細については、以下を参照してください](https://support.office.com/article/73660701-018b-4961-bc28-6976cf9ebfb3)。

## <a name="insert-shapes"></a>図形の挿入

Web スプレッドシートの Excel に、ボックス、円、線、コネクタ、矢印などの図形を追加して書式設定します。 図形の追加について [詳しくは、次のページを参照してください](https://support.office.com/article/0e492bb4-3f91-43b5-803f-dd0998e0eb89#OfficeVersion=Web)。

## <a name="keyboard-shortcuts"></a>キーボード ショートカット

キーボード ショートカットを制御して、デスクトップ版と Web 版の Excel で同じように動作する場合は、[キーボード ショートカット] 設定を変更すると、ブラウザーのショートカットを上書き &mdash; することもできます。 [ヘルプ キーボード **ショートカット**  >  **] を選択します**。
  
## <a name="merge-cells"></a>Merge cells

Use Merge &amp; Center on the Home tab to combine and center the contents of the selected cells in one larger cell. 配置を変更するには、[配置] ボタンを選択します。

## <a name="named-ranges"></a>名前付き範囲

Web 用の Excel で名前付き範囲を作成することはできませんが、Excel デスクトップで作成した名前付き範囲は、Excel の Excel for the web で使用できます。 名前付き範囲を選択すると、数式バーの直前に名前が表示されます。

## <a name="number-formatting"></a>数値の書式

数値の書式を変更します。書式の一覧での [通貨] や [短い日付形式] などの書式の選択、小数点以下の桁数の変更、または桁区切り記号の追加を行います。
  
## <a name="office-add-ins"></a>Office アドイン

 一部のアドインOfficeは、Web 用の Excel で使用できます。 Excel 用のアドインは、Office ストアから入手できます。 Excel アドインを [取得する方法について詳しくは、以下を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271667)。

## <a name="offline-viewing-and-authoring"></a>オフラインでの表示と編集

Excel for the web は、インターネット ブラウザーから起動され、インターネット接続に依存します。 オフラインでスプレッドシートにアクセスするには、ご使用のコンピューターに Microsoft Excel デスクトップ アプリをインストールし、Excel デスクトップ アプリを使用して Excel ワークシートの表示や編集を行う必要があります。

## <a name="pivottables"></a>PivotTables

Web スプレッドシートのピボットテーブルを Excel に挿入し、データの計算、集計、分析を行います。 Web 用の Excel でピボットテーブルを [挿入する方法の詳細については、以下を参照してください](https://support.office.com/article/A9A84538-BFE9-40A9-A8E9-F99134456576#OfficeVersion=Web)。

## <a name="post-to-social-network-and-present-online"></a>ソーシャル ネットワークへの投稿とオンライン共有

ソーシャル ネットワーク ページ、Skype for Business の会話、または会議に Web 上のスプレッドシートの選択した部分を埋め込んで共有します。 Excel for the web では、これらの機能はサポートされていません。
  
## <a name="power-pivot-viewing"></a>Power Pivot の表示

Excel for the web では、Power Pivot テーブルとグラフを表示できますが、Power Pivot データ モデルを作成するには Excel デスクトップ アプリが必要です。
  
## <a name="power-view-viewing"></a>Power View viewing

ブック内のシートを表示Power View操作できますが、Excel を使用して web Power Viewを作成することはできません。 Power View を作成するには、デスクトップ アプリケーションを使用する必要があります。 Power View では、対話型で、プレゼンテーションに対応したビューでデータの作成、共有、探索ができます。 1 つのワークシートの相互にリンクされた対話型のマップ、グラフ、テーブルで情報を表示します。 詳細については、「Power View」 [を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271674)。
  
## <a name="print"></a>印刷

ワークシート全体または現在の選択範囲をプリンターに送信するには、Web 用の Excel の [印刷] コマンドを使用します。 Excel for [the web で印刷する方法の詳細については、以下を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271676)。
  
## <a name="real-time-co-authoring"></a>リアルタイム共同編集

2 人以上のユーザーが、Excel デスクトップ アプリの代わりに各自の Web ブラウザーを使用して、同時に同じスプレッドシートで作業することができます。リアルタイム プレゼンスを使用すれば、共同編集者が文書のどこを編集しているかを確認して編集中の競合を避けたり、彼らの変更箇所を表示したりできます。リアルタイム共同編集の詳細については、「[Excel ブックの共同編集を使用して同時に共同作業を行う](https://go.microsoft.com/fwlink/?linkid=850793)」を参照してください。
  
## <a name="recommended-chart-creation-and-editing-with-formatting-controls"></a>おすすめグラフの作成と書式設定を使用した編集

高度なグラフは、Microsoft Excel デスクトップ アプリのみでサポートされます。 Excel for the web では、これらの機能はサポートされていません。 使用可能なグラフの [種類について詳しくは、以下をご覧ください](https://go.microsoft.com/fwlink/p/?LinkId=271666)。
  
## <a name="refresh-existing-data-connections"></a>Refresh existing data connections

ブック内のデータがデータベース、Web ページ、分析キューブなどの外部ソースに接続されている場合は、Excel for the Web を使用してそのデータを更新できます。

## <a name="rename-and-add-sheets"></a>Rename and add sheets

[新しいシート] アイコンを選択してシートを追加します。 シートの名前を変更するには、シートのタブを右クリックします。
  
## <a name="rename-file-while-workbook-is-open"></a>ブックが開いている間にファイルの名前を変更

Web の Excel では、ヘッダーでファイル名を選択し、新しい名前を入力することで、ブック ファイルの名前を閉じることなく名前を変更できます。 Excel デスクトップ アプリを使用してファイルの名前を変更するには、デバイス上のその場所に移動して、その名前を変更します。
  
## <a name="replace"></a>置換

Web 用の Excel では使用できません。 お使いのコンピューターに Excel デスクトップ アプリがインストールされている場合は、Excel を使用してドキュメントの内容の検索と置換を行うことができます。 Excel で検索と [置換を使用する方法の詳細を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=282333)。
  
## <a name="rights-management-irm-and-password-security"></a>アクセス権管理:IRM とパスワード セキュリティ

 Excel デスクトップ アプリでは、パスワード、アクセス許可、その他の制限を使ってブックを保護します。 Excel for the web では、シート保護を含むブックを読み込み、操作できます。このため、ユーザーは保護されたセルを選択または入力できます。 これにより、ユーザーは保護されたセルの選択や入力ができなくなります。[Excel ブックの保護](https://go.microsoft.com/fwlink/p/?LinkId=271672)の詳細をご確認ください。
  
## <a name="save-or-download-a-copy"></a>Save or Download a copy

ブックを自分用にコピーする必要がある場合は、[ファイル] タブの [名前を付けて保存] コマンドを使用するか、またはブックのフォルダーでブック名を右クリックして [ダウンロード] コマンドを使用し、お使いのコンピューターにブックをダウンロードします。

他のユーザーがドキュメントを表示または編集できるように、ワンクリックでリンクを送付できます。 [365](https://go.microsoft.com/fwlink/p/?linkid=272770)で web Officeを使用して一緒に動作する方法Officeします。
  
## <a name="sheet-protection"></a>シートの保護

Excel ファイルを他のユーザーと共有する場合、ワークシートが変更されるのを防ぐためにワークシートを保護することができ、ワークシートのユーザーに許可する操作を選択することもできます。

## <a name="sheet-views"></a>シート ビュー

[シート ビューを](https://support.microsoft.com/office/sheet-views-in-excel-0eea3dc5-d7d1-44c5-a953-25ebfbd6c1a6) 使用すると、他のユーザーによって中断されることなく、Excel ワークシートのカスタマイズされたビューを作成できます。 たとえば、他のユーザーによるドキュメントの並べ替えやフィルター処理の影響を受けることなく、重要なレコードのみを表示するフィルターを設定できます。 同じワークシートに複数のシート ビューを設定することもできます。
  
## <a name="slicers"></a>スライサー

ワークシートのスライサーを表示し、スライサー ボタンを選択してデータをフィルター処理します。 Web 用の Excel でスライサーを作成または編集することはできませんが、既存のスライサーは削除できます。
  
## <a name="sort-and-filter-data"></a>Sort and filter data

Excel for the web では、すべてのワークシートの表示と並べ替え、データの並べ替えとフィルター処理 (上から下または左から右、複数のレベル)、ピボットテーブルの詳細の詳細を確認できます。 アイコンまたは色の値を使用して、アイコン セットまたはカラー スケールを使用する条件付き書式のデータを並べ替える。 データの並べ [替えとフィルター処理の詳細を確認します](https://support.office.com/article/62d0b95d-2a90-4610-a6ae-2e545c4a4654?ad=US#ID0EAADAAA=Web)。

Microsoft Excel デスクトップ アプリでは、ワークシートの検査と比較の機能を使用して、ブックのバージョンの比較、ブックの問題点や不整合の分析、またはブックとワークシートとの間のリンクの確認を行うことができます。 Excel for the web では、これらの機能はサポートされていません。 スプレッドシート照会で [実行できる操作の詳細については、以下を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271677)。

アンケートを使用して、記入できるユーザーに Web フォームの Excel を送信します。応答は、オンライン ワークシートで自動的にコンパイルされます。 Excel のアンケート [について詳しくは、次のリンクを参照してください](https://go.microsoft.com/fwlink/p/?LinkId=391470)。

## <a name="vba-and-macro-scripting"></a>VBA およびマクロの作成

Microsoft Excel デスクトップ アプリは、データの操作、分析、共有に使用できる強力なツールです。 これには充実した機能が備わっていますが、必要な場合には、プログラミング言語である VBA (Visual Basic for Applications) を使用して、日常的な繰り返し作業を実行するマクロやユーザー インターフェイス (UI) では対応していないような作業を実行するマクロを作成して、作業を効率的に進めることができます。 Excel for the web では VBA を使用してマクロを作成することはできませんが、ファイルに含まれる VBA を削除 (または破損) することなく、VBA 対応のスプレッドシートを開いて編集できます。 マクロの使用 [を開始する方法の詳細を参照してください](https://go.microsoft.com/fwlink/p/?LinkId=271678)。
